Target Audience:


The main audience for this project would be school-age children, most likely primary school age, who are just beginning to interact with broader topics in school besides the fundamentals. The target audience most likely have little to no experience with these artistic subjects, and need to be versed in the basic in order to interact with the end result. Ultimately, they will be able to grow from these projects, and maybe even expand on them.





Goals:


The first goal of this project is to build a section in which they will be able to create their own work based on what they learned. Hopefully, it will increase from only 3 sections, but originally that will be the goal point. The next step will be to use a database library such as firebase, in order to store these projects for others to see. Finally, I will have a learning section so they can learn more about these subjects. Each plan will be building on the others.
